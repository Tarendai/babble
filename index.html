Babble

WordPress multi-language plugin

Download this project as a .zip file Download this project as a tar.gz file

A plugin to handle translating content into a variety of languages. The plugin was built with an aversion to both additional database tables, additional columns or column changes and a desire to keep additional queries to a minimum.

There are a lot of @FIXME comments, expressing doubts, fears, uncertainties and unknowns; feel free to weigh in on any of them.
